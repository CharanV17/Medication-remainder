Medication Reminder Frontend (Simple UI)
---------------------------------------

How to run:

  cd frontend
  npm install
  npm install -D tailwindcss postcss autoprefixer
  npx tailwindcss init -p
  # ensure tailwind.config.js contains ./index.html and ./src/**/*.{js,jsx} as content
  npm run dev

Backend: make sure your backend is running at http://localhost:3000
